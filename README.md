<h3>Movement</h3>
<p>Use left click for pressing button and use f for hitting the ball</p>
<p>There is no win condition but you can loose by returning to first press scene</p>
<p>This game is called pinball</p>
<p>One issue you may encounter is weird bouncing and you may also get the game button not seem to work but just wait for a few seconds and the button will work</p>
<p>You can play this on itch.io</p>
